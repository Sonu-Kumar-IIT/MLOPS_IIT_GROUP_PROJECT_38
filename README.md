## Quick Start

### 1. Install dependencies

```bash
pip install -r requirements-dev.txt
```

### 2. Set environment variables

```bash
export HF_TOKEN="<your-huggingface-token>"
export WANDB_API_KEY="<your-wandb-api-key>"
export HF_REPO_ID="<your-hf-username>/mlops-group38-ner"
```

### 3. Run the full training pipeline

```bash
python -m src.training.main
```

### 4. Run inference from Hugging Face Hub

```bash
INPUT_TEXT="Apple is looking at buying a U.K. startup for $1 billion." \
    python -m src.inference.infrence_from_hub
```

### 5. Build and run with Docker

```bash
# Build
docker build --build-arg HF_MODEL_NAME=<your-hf-username>/mlops-group38-ner \
    -t mlops-group38-ner .

# Run
docker run --rm \
    -e HF_TOKEN="<your-token>" \
    -e INPUT_TEXT="Google launched a new model in California." \
    mlops-group38-ner
```

## Dataset

- **Name**: `yongsun-yoon/open-ner-english` (Hugging Face Datasets)
- **Task**: Token-level Named Entity Recognition
- **Size**: 2,000 train / 400 validation (CPU); 5,000 / 1,000 (GPU)

### Data Cleaning

| Issue | Action | Reason |
|---|---|---|
| YAML front-matter | Stripped with regex `---…---` | Noise from markdown docs |
| Citation markers `[@...]` | Removed | Bibliography artefacts |
| LaTeX commands `\cmd` | Removed | Non-natural-language tokens |
| Non-alphanumeric chars | Replaced with space | Clean token boundaries |
| Multiple whitespace | Collapsed to single space | Normalise spacing |
| Token–label length mismatch | Row dropped | Prevents misaligned labels |
| Rare entity types | Mapped to `O` tag | Avoids training on sparse classes |

## Model

**Base model**: [`dslim/bert-base-NER`](https://huggingface.co/dslim/bert-base-NER)

Selected because:
- Cased architecture (capitalisation is a strong NER signal)
- ~91.3 F1 on CoNLL-2003 benchmark
- Broader label coverage vs. distilbert alternative
- Fits within Kaggle free GPU tier (~110M parameters)

## Training Experiments

| Hyperparameter | V1 (run-v1) | V2 (run-v2) |
|---|---|---|
| Learning rate | 3e-5 | 5e-5 |
| Epochs | 3 | 5 |
| Batch size | 16 | 16 |

| Warmup ratio | 0.1 | 0.1 |

| Metric | V1 | V2 | Better |
|---|---|---|---|
| Accuracy | 0.8486 | 0.8522 | V2 |
| Macro F1 | 0.1296 | 0.1153 | **V1** |
| Weighted F1 | 0.8307 | 0.8305 | **V1** |
| Macro Precision | 0.1615 | 0.1391 | **V1** |
| Macro Recall | 0.1381 | 0.1252 | **V1** |

**V1 selected** for deployment (lr=3e-5, 3 epochs).
